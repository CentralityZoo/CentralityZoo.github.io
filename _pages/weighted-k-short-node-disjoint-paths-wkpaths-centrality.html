Weighted k -short node-disjoint paths (WKPaths) centrality


The weighted \(k\)-short node-disjoint paths (WKPaths) centrality was introduced by White and Smith [2] as a variant of closeness centrality. Rather than considering all shortest paths between nodes, WKPaths considers the set of \(k\)-short paths, defined as all paths of length at most \(k\) with no shared intermediate nodes. The WKPaths centrality of a node \(i\), denoted \(c_{WKPaths}(i)\), is defined as
\begin{equation*}
c_{WKPaths}(i) = \frac{1}{N} \sum_{j \in \mathcal{N}} \sum_{P \in \mathcal{P}_k(j,i)} δ^{-|P|},
\end{equation*}
where \(\mathcal{P}_k(j,i)\) is the set of \(k\)-short paths from node \(j\) to node \(i\), \(|P|\) is the length of path \(P\), and \(δ\) is a scalar weighting factor with \(1 \leq δ \leq \infty\) (e.g., \(δ = 2\)). Shorter paths contribute more heavily to the centrality due to the exponential decay factor \(δ^{-|P|}\).
