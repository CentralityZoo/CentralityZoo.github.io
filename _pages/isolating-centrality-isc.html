--- title: Isolating centrality (ISC) permalink: /isolating-centrality-isc/ search: true ---

Isolating centrality (ISC) identifies nodes that critically affect network connectivity [2]. The ISC of node \(i\), denoted \(c_{\text{ISC}}(i)\), is defined as\begin{equation*}c_{\text{ISC}}(i) = |\mathcal{N}(i) \cap d_{δ}| \cdot |\mathcal{N}(i)|,\end{equation*}where \(\mathcal{N}(i)\) is the set of neighbors of node \(i\), and \(d_{δ}\) is the set of nodes with minimal degree \(δ = \min_j |\mathcal{N}(j)|= \min_j d_j\). A high ISC value indicates that the node lies between densely connected internal nodes and sparsely connected terminal nodes. Removing such central nodes can weaken or even disrupt communication paths between groups of active nodes.
